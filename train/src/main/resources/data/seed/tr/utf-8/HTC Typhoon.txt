HTC Typhoon HTC tarafından Ağustos 2004 yılında üretilen bir Windows Mobile cep telefonu modelidir. Microsoft'un Windows Mobile 2003 SE işletim sistemini çalıştırıyor, ancak bir sürümü internete sızdırdıktan sonra Windows Mobile 5.0'ı da çalıştırabiliyor. Java uygulamalarını destekler. Buna ek olarak, Windows Mobile 6.0, Windows Mobile 6.1 ve Windows Mobile 6.5'in saldırıya uğramış veya "pişmiş" sürümleri internette dolaşıma girmişdir.Typhoon'un yapıldığı dönemde HTC, son kullanıcılara cihaz satma işinde değildi. Bunun yerine, şirketin cihazlarını yeniden dağıtacak birçok ortak vardı.
200 MHz'de çalışan ARM Texas Instruments OMAP 730 işlemcisine dayanıyor. 32 MB dahili RAM'a ve 64 MB flash ROM'a sahiptir ve miniSD yuvası ile genişletilebilir. 176x220 çözünürlükte 65.536 renk ile TFT ekrana sahiptir.
